# Image Gallery
Built using Angular and Node.

Run below commands to run the application after ``git clone https://github.com/rohitkalva/image-gallery.git``. Front-end runs on ``localhost`` port ``4200`` and server runs on port ``5000``

## Gallery
``cd gallery``
``npm install``
``ng serve``

## Server
``cd server``
``npm install``
``node index.js``
